# PyRay
Pong made with library called PyRay which is a binding of Raylib library

## Upgrade PIP
#### MAC/LINUX
```bash
    python3 -m pip install --upgrade pip
``` 

#### WINDOWS
```bash
    python -m pip install --upgrade pip
```

## Install Pyray
#### MAC/LINUX
```bash
    python3 -m pip install pyray
```

#### WINDOWS
```bash
    python -m pip install pyray
```
## Install Pygame
#### MAC/LINUX
```bash
    pip3 install pygame
```
#### WINDOWS
```bash
    pip install pygame
```

## RUN
#### MAC/LINUX
```bash
    cd PyRay
    python3 ./src/main.py
```

#### WINDOWS
```bash
    cd PyRay
    python ./src/main.py
```
